L'Origine

Tout commence par une amitié de fer entre trois lycéens de 17 ans. Entre les cours, un constat nous frappe : les commerçants locaux sont les piliers de notre ville, mais ils sont invisibles sur le web.

La Mission

Le Relais Web est né pour briser ce plafond de verre. Nous utilisons le Code Pur pour offrir aux artisans la puissance technologique des plus grandes entreprises.
